# 关于角色和用户的操作

## mongosh命令

```mongosh
# 角色：
# 列出全部角色：
show roles
# 新建角色

# 删除角色

# 用户：
# 列出全部用户：
show users
# 新建用户
db.createUser({
  user: "test_admin",
  pwd: "abc1234",
  roles: [
    { role: "userAdmin", db: "test" },
  ],
  comment: "test库管理员"
})
db.createUser({
  user: "test_rw",
  pwd: "abc1234",
  roles: [
    { role: "readWrite", db: "test" },
  ],
  comment: "test库读写者"
})
# 删除用户
db.dropUser("test_admin")

# 改变用户的角色

```
