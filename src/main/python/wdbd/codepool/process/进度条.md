## Bar条

进度条，分别有构造函数、next()和finish()三个主要函数，每执行一次next()进度条推进一步。

不同的Bar有不同的样式：

- ChargingBar
- FillingSquaresBar
- FillingCirclesBar
- IncrementalBar
- PixelBar
- ShadyBar

实例代码：

```python
from progress.bar import Bar
import time

action_list = ['A', 'B', 'C', 'D', 'E']
bar = Bar('Processing', max=len(action_list))
for action in action_list:
    time.sleep(1)
    bar.next()
bar.finish()
```

## 转轮条

进度条，分别有构造函数、next()和finish()三个主要函数，每执行一次next()进度条推进一步。

```python
from progress.spinner import Spinner
import time

spinner = Spinner('Loading ')
state = "START"
idx = 0
while state != 'FINISHED':
    # Do some work
    time.sleep(1)

    if idx == 10:
        state = "FINISHED"
    idx += 1
    spinner.next()
spinner.finish()
```
